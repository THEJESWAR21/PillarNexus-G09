Technical Description

Home Page

I created a dynamic homepage with a jumbotron banner using a full-width background image via background-image in CSS. The navigation bar is placed at the top and styled using Flexbox and hover transitions to provide interactivity.

Table Page

The table includes a logo, header thumbnails, colored rows, icon symbols, and merged cells. I used colspan, rowspan, :hover, and alternating row colors with :nth-child(). Tooltip effects were added using the title attribute and ARIA labels for accessibility.

Content Page

The content page was divided into sections using <section> tags. Internal navigation was created using anchor tags and id attributes. A fixed "Go to Top" button was implemented using position: fixed in CSS.

Challenges and Lessons Learned

One of the main challenges was ensuring internal navigation links worked properly across all screen sizes. I resolved this by testing anchor behavior and adjusting padding and scroll-margin-top to account for the fixed navbar. I also learned how to use Flexbox and grid systems for responsive layouts.

Accessibility

All images include alt text for screen readers. Navigation uses semantic HTML5 elements like <nav> and <main>, and buttons use aria-labels where necessary. The color contrast was checked for readability.

Compliance

The pages follow Janet web publishing guidelines by using accessible elements, ensuring responsiveness, using validated HTML/CSS, and avoiding copyrighted material. All pages validate with the W3C validation service.
